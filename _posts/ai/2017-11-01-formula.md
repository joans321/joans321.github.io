# Markdown数学公式


* 数学公式起始和结尾标志

数学公式以 `$` 开头和结尾，例如: `\$f(x) = x^2 + 1\$` 显示为: $f(x) = x^2 + 1$

如果需要独占一行的话，则以 `$$` 开头和结尾。 例如: `\$\$f(x) = a + bx\$\$` 显示为: $$f(x) = a + bx$$



----




* 符号上标和下标

上表用 `^` 表示，下标用 `_` 表示。

例如 `\$f(x) = a_0 + a_1 * x + a_2 * x^2\$`  显示为:  $f(x) = a_0 + a_1 * x + a_2 * x^2$
